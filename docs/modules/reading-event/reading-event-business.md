## 🎯 业务目标

### 核心目标
创建一个结构化的共读活动平台，通过科学的激励机制促进用户的持续阅读和深度思考。

### 业务价值
- **用户价值**: 提供结构化阅读体验，培养阅读习惯
- **社区价值**: 建立学习型社区，促进知识分享
- **商业价值**: 通过会员制和文化基金模式实现可持续运营

## 🔄 核心业务流程

### 流程1：活动创建
```
1. 小组长确定书目和活动模式
2. 规划日程和阅读进度（建议 20-30 页/天）
3. 设置费用模式（免费/押金制/收费制）
4. 发布活动，开放报名（默认25人，最多50人）
5. 用户报名或围观
6. 召集领读人占位
7. 活动正式开始
```

### 流程2：每日共读循环
```
1. 当日领读人发布领读内容（使用模板）：
   - 阅读进度建议
   - 2-3 个领读问题
   - 打卡要求和截止时间

2. 共读人在 24 点前完成打卡：
   - 笔记内容 ≥ 100 字
   - 与主题相关，不得复制粘贴

3. 次日领读人浏览打卡作业：
   - 选出 1-3 朵小红花
   - 转发到微信群

4. 轮换到下一位领读人
```

### 流程3：活动结束
```
1. 小组长统计打卡完成情况
2. 统计小红花获得者（前 3 名）
3. 生成参与证书（完成证书、小红花前三名证书、自定义证书）
4. 结算费用：
   - 押金制：退还达标者押金池金额，小组长获得固定报酬
   - 收费制：全部费用作为小组长报酬
   - 免费制：无费用结算
5. 活动总结和归档
```

## 🎨 活动模式设计

### 模式1：笔记打卡方式
**核心特点**:
- 每日强制打卡机制
- 字数要求: 至少100字
- 领读人点评和发放小红花
- 支持周末休息设置

**适用场景**:
- 需要强制自律的用户
- 初期读书会
- 重视完成率的群体

### 模式2：自由讨论方式
**核心特点**:
- 无强制打卡要求
- 开放式讨论平台
- 简化管理流程
- 适合经验丰富的用户群体

**适用场景**:
- 自律性强的用户群体
- 高级读书会
- 重视深度讨论的群体

### 模式3：视频会议方式
**核心特点**:
- 定期视频会议讨论
- 深度面对面交流
- 时间由小组长直接设定
- 支持后期修改会议时间

**适用场景**:
- 需要深度交流的群体
- 重要书籍精读
- 小规模精品活动

### 模式4：线下交流方式
**核心特点**:
- 定期线下聚会讨论
- 面对面深度交流
- 时间由小组长直接设定
- 支持后期修改聚会时间

**适用场景**:
- 同城用户群体
- 需要深度社交的读书会
- 重要主题研讨

## 💰 费用机制设计

### 免费模式
**特点**:
- 无任何费用要求
- 零门槛参与
- 适合试水和体验

**适用场景**:
- 小规模朋友聚会
- 测试性活动
- 公益性读书会

### 押金制模式
**费用设置**:
- 范围: 1-500元
- 默认: 100元
- 由小组长根据活动规模设置
- 报名时收取费用

**费用分配机制**:
```
总费用: 100元 (示例)
├─ 小组长固定报酬: 20元 (20%)
└─ 参与押金池: 80元 (80%)
   └─ 达标者: 按完成率退还押金池金额
```

**退还机制**:
```
完成率 ≥ 设定标准: 退还80%押金池金额
完成率 < 设定标准: 不退还押金池金额
```

**特殊说明**:
- 小组长固定获得20%作为组织报酬，不参与退还分配
- 80%费用进入押金池，根据完成率进行退还
- 简单粗暴的退还机制，无部分退还
- 任何人无法代替打卡，必须亲自完成

### 收费制模式
**费用设置**:
- 范围: 1-500元
- 默认: 100元
- 由小组长根据活动价值设置
- 报名时收取费用

**费用分配机制**:
```
总费用: 100元 (示例)
├─ 小组长报酬: 100元 (100%)
└─ 平台服务费: 0元 (暂不收取)
```

**特点**:
- 收费不退，全部费用归小组长所有
- 适合高质量、有附加值的活动
- 小组长需要提供更多价值服务

**适用场景**:
- 专业导师带领的深度阅读
- 包含额外资源和服务的活动
- 小规模精品读书会
- 特殊主题或稀缺资源活动

## 🏆 激励机制设计

### 小红花系统
**发放规则**:
- 每日领读人可发放1-3朵小红花
- 基于打卡质量和思考深度选择
- 发放后有记录和公示

**统计功能**:
- 个人小红花排行榜
- 活动小红花统计
- 历史小红花记录

### 证书系统
**证书类型**:
- **完成证书**: 达到完成率标准的所有参与者
- **小红花前三名证书**: 活动结束后小红花数量最多的前三名
- **自定义证书**: 小组长可自由设定证书名称和获得者

**证书特点**:
- 独一无二的证书编号
- 包含个人成就数据
- 支持分享和展示
- 可设置公开/私密状态

## 🎯 会员制替代方案（未来规划）

### 问题思考
"读书人的事怎么跟钱这么息息相关呢？"

### 核心理念
将付费模式从"单次交易"转向"长期归属"，减少读书活动的商业感，增强文化交流的氛围。

### 年度会员制设计
**会员类型**:
```
📚 年度会员 (365天)
  ├─ 免费参与所有共读活动 (无额外费用)
  ├─ 优先报名热门活动
  ├─ 专属会员徽章和身份标识
  ├─ 参与会员专属活动和讨论
  └─ 年费: 365元/年 (平均1元/天)

🌱 普通用户
  ├─ 可围观所有活动讨论
  ├─ 按活动付费参与 (现有收费制)
  ├─ 可购买单次活动参与资格
  └─ 费用: 按具体活动标准
```

### 文化基金模式 (折中方案)
**重新定义费用概念**:
```
文化共建基金 (而非"费用"或"收费")
├─ 基金用途: 支持小组长组织活动
├─ 基金管理: 透明公开，用于社区建设
├─ 基金精神: 共建共享的文化平台
└─ 基金回报: 更优质的活动内容和体验
```

**界面和文案调整**:
- "付费参与" → "共建文化基金"
- "收费" → "基金贡献"
- "小组长报酬" → "活动组织支持"
- "退款" → "基金返还"

### 实施路线图
**第一阶段 (3个月)**:
- 文化基金模式试点
- 文案和界面调整
- 用户反馈收集

**第二阶段 (6个月)**:
- 年度会员制内测
- 会员权益设计
- 定价策略确定

**第三阶段 (12个月)**:
- 会员制全面推广
- 双轨制运行
- 数据评估和优化

- 会员转化率: ≥ 15%
